# Employee Management System

## Description
Console application for employee management developed in Java.

## Features
- Add employees
- View employee list
- Search employee by ID
- Delete employees
- Save data to file
- Load data after restart

## Technologies
- Java
- OOP
- ArrayList
- Scanner
- File I/O

## Project Structure
- Employee.java
- EmployeeManager.java
- Main.java

## How It Works
1. Add a new employee
2. View all employees
3. Search employee by ID
4. Delete employee
5. Save data and exit

## Skills Demonstrated
- Object-Oriented Programming
- Working with Collections
- File Handling
- User Input Processing
- Data Persistence
